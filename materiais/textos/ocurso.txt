Curso de HTML5 e CSS3
Confira o que você vai aprender!

Aprenda no curso de HTML os conceitos essenciais para a criação de páginas web, junto com os novos recursos e possibilidades que o HTML5 e CSS3 oferecem para a criação de sites com códigos limpos e estruturados com base nas melhores práticas de programação do mercado.
					

Objetivos

O curso proporciona ao aluno o conhecimento necessário para a criação de sites através de codificação HTML e CSS, utilizando o editor Sublime Text.

Pré-Requisitos

Sistema operacional, Windows ou Mac OS X;
Gerenciar arquivos, criar pastas, renomear arquivos, etc.


A quem se destina

O curso de HTML 5 e CSS3 do DRC, realizado em São Paulo, é destinado a pessoas que desejam criar web sites sem depender de nenhuma ferramenta visual, ou ainda se tornarem desenvolvedores de aplicações web.
